# Zed for Ed demo

1. Ask the AI to setup your Dockerfile/`devcontainer.json` dependencies
2. In the Command Palette,  (`Cmd + p > project: open in dev container`) or click the toast on the bottom right. This should install the proper dependencies needed for your classroom. In this case, that's `numpy` and `matplotlib` via `uv`.
3. Run 

```bash
python3 pirates_vs_co2.py
```

and click on the PNG image.
